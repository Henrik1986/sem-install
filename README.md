# SEM Installer

Denna integration används för att ladda ner filerna för energisystemet första gången.

## Installation

1. Installera denna integration via HACS
2. Starta om Home Assistant
3. Gå till **Inställningar → Integrationer**
4. Lägg till **SEM Installer**
5. Klicka på Install för att ladda ner paketet

## Viktigt

- Installationen sker i två steg:
  1. Installera via HACS
  2. Lägg till integrationen i Home Assistant

- När detta är gjort laddas alla filer ner automatiskt

## Efter installation

- Följ instruktionerna i GitHub-repot
- När systemet är igång kan denna integration tas bort
- Systemet uppdateras därefter via energisystemets Hemvy

⚠️ Detta är en engångsinstallation
